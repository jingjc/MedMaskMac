# Privacy Policy

MedMaskMac is designed as a local-first redaction tool. PDFs and images are intended to be processed locally on the user's Mac. By default, the project does not require uploading documents to a remote server for redaction.

Documents processed by MedMaskMac may contain names, phone numbers, ID numbers, medical record numbers, inpatient numbers, addresses, medical details, and other sensitive information.

OCR and automatic redaction can make mistakes. Users should manually review all redactions before sharing exported files.

Do not submit real personal, medical, financial, or identity documents in GitHub issues, pull requests, screenshots, or test fixtures. Use only synthetic or fully redacted samples.
