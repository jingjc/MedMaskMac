# Security Policy

MedMaskMac is currently early alpha software.

## Reporting a Vulnerability

Report security and privacy issues privately. Do not open a public issue containing sensitive details.

Do not include real personal, medical, financial, or identity documents in a report. Synthetic or fully redacted test files are acceptable.

Security and privacy reports may include:

- Sensitive text not being redacted correctly
- Redaction boxes appearing in preview but missing from export
- Exported files retaining hidden sensitive text
- Temporary files leaking private document contents
- Logs containing sensitive OCR results
- Test fixtures containing real personal or medical information

Valid reports will be reviewed as soon as reasonably possible. No fixed response time is guaranteed.
